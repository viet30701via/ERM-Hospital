## Kiến trúc hệ thống (Architecture)

Dự án được xây dựng theo mô hình Client-Server:

- **Frontend (Client):** React / NextJS
- **Backend (API):** NestJS (chạy trên Node.js)
- **Database (DB):** MongoDB

## Ghi chú thực hành Ngày 1

- Đã quan sát Network tab trên Chrome DevTools.
- Đã thử gửi GET request bằng Postman/curl.
